# Price Tag Scanner

Сервис на Streamlit, который принимает видео с ценниками, прогоняет каждый кадр через YOLO-детектор и пытается считать QR-код с каждого найденного ценника. На выходе — CSV с координатами боксов и данными QR.

## Что нужно перед запуском

- Python 3.10+
- conda или venv
- CUDA-совместимая видеокарта (опционально, но желательно)
- Обученные веса модели — файл `best.onnx` должны лежать в папке `models/`

## Установка

Создаём окружение и ставим зависимости:

```bash
conda create -n VilenaProject python=3.10
conda activate VilenaProject
pip install -r requirements.txt
```

## Запуск сервиса

```bash
streamlit run tools/video_scanner.py
```

Откроется браузер на `http://localhost:8501`.

## Как пользоваться

1. Загрузи видео (.mp4 или .mov)
2. Настрой шаг между кадрами — чем меньше, тем дольше обрабатывается, но больше шанс поймать каждый ценник
3. Порог уверенности по умолчанию 0.7
4. Нажми "Запустить сканирование"
5. После завершения скачай CSV кнопкой внизу

## Формат выходного CSV

Файл содержит одну строку на каждый найденный ценник. Заполняются автоматически:

- `filename` — имя исходного видео
- `frame_timestamp` — время кадра в миллисекундах
- `x_min`, `y_min`, `x_max`, `y_max` — координаты бокса
- `qr_code_barcode` — данные QR-кода если удалось считать

Остальные поля (`product_name`, `price_default`, `barcode` и т.д.) оставлены пустыми — под заполнение OCR (в разработке).
